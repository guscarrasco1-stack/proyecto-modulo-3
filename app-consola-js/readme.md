# Aplicación de Consola en JavaScript

Proyecto Módulo #3– Fundamentos de Programación en JavaScript.

#Descripción
Esta aplicación permite realizar operaciones matemáticas básicas y trabajar con estructuras fundamentales del lenguaje JavaScript, como condicionales, ciclos, funciones, arreglos y objetos.

#Funcionalidades
- Suma, resta, multiplicación y división
- Validación de entradas del usuario
- Uso de menú interactivo
- Filtrado de usuarios mayores de edad
- Ejecución en la consola del navegador

#Tecnologías utilizadas
- JavaScript (ES6)
- Consola del navegador

#Cómo ejecutar
1. Abrir el archivo `index.js`
2. Copiar el código
3. Pegar en la consola del navegador
4. Presionar Enter

#Estructura del proyecto
- index.js → Código principal
- README.md → Documentación
- capturas/ → Evidencia visual del funcionamiento
